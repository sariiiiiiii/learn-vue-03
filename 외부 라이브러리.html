-- 외부 라이브러리 모듈화 - 이유 : Vue.js 관련 라이브러리가 없을 때 일반 라이브러리를 결합할 수 있어야 함 - 종류 1) 차트 2) 데이트 피커 3) 테이블 4) 스피너 -- 설치 과정 1. 차트 라이브러리 NPM으로 설치 2. 설치된 라이브러리를 import로 App.vue에서 로딩 3. mounted() 라이프 사이클 훅에서 차트를 그림 4. 차트를 컴포넌트화 - LineChart와 BarChart를 컴포넌트화를 했는데 두 컴포넌트의 canvas 태그의 id값이 동일해서 충돌하는 문제 생김 충돌문제를 방지하기 위해 컴포넌트화를 해서 같이 등록해서 사용할 때 태그의 id값 중복이 되지 않게 조심 - ref(레퍼런스) 활용(DOM에 접근) 5. 컴포넌트의 플러그인화(plugins -> ChartPlugins.js) => https://vuejs.org/guide/reusability/plugins.html - 인스턴스가 생성됐을 때 모든 컴포넌트에서 사용하고 싶은 기능, 라이브러리도 하나의 플러그인이 될 수 있다(ex VueRouter, Vuex) - Vue.use() 형태는 플러그인을 사용한 형태 6. 컴포넌트 통신을 이용한 차트 컴포넌트 기능 결합
